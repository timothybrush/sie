# Home

A private, self-hosted knowledge vault. Documents are turned into clean, linked markdown by open models on our own GPUs (via the SIE MCP edge). Raw documents and personal data never leave our infrastructure; the model only ever sees the small markdown.

- [[Contracts]] - index of every contract, agreement, NDA, and invoice
- `Entities/` - companies and people, auto-linked from the notes
- `Contracts/` - one note per document, each linking its parties and people

This vault already holds a web of standing agreements across ten organizations. Drop a document into `inbox/` and ask the agent to add it; watch it connect into the graph.
